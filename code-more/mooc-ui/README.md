# mooc-ui

这是一个跟着慕课网学习搭建的组件库。

# 一、搭建vue项目

```shell
vue create mooc-ui
```

# 二、项目目录介绍

```
|-- node_modules  项目依赖文件
|-- public  静态文件
|-- src 业务逻辑
  |-- main.js 入口文件
|-- .gitignore  git忽略文件配置
|-- babel.config.js  babel配置文件  
|-- package.json  记录项目开发依赖
|-- package-lock.json  记录依赖版本号
|-- README.md  说明文档
```
